- [FiraCode NerdFont](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.1.1/FiraCode.zip) for lualine
- [LazyGit](https://github.com/jesseduffield/lazygit?tab=readme-ov-file#ubuntu) for toggleterm
